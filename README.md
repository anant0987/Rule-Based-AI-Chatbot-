#  Rule-Based AI Chatbot

A simple yet engaging rule-based chatbot project built in Python. It demonstrates how a chatbot can respond to predefined questions, remember previous conversations, learn new answers, and save information for future sessions.

## ✨ Features

- Responds to common greetings and questions
- Supports simple conversation flow with basic commands
- Keeps chat history during a session
- Learns new responses when a user teaches it something new
- Saves memory permanently using JSON files

## 📁 Project Files

- [Chatbot.Py](Chatbot.Py) - Basic rule-based chatbot with simple responses
- [chatbot2.py](chatbot2.py) - Adds chat history and learning capability
- [chatbot3.py](chatbot3.py) - Adds permanent memory storage using a JSON file
- [README.md](README.md) - Project documentation

## ▶️ How to Run

Make sure Python is installed on your system, then run one of the chatbot files:

```bash
python Chatbot.Py
```

```bash
python chatbot2.py
```

```bash
python chatbot3.py
```

## 💬 Example Commands

## 🛠️ Technologies Used

- Python
